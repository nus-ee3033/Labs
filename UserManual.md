## Steps to use this template

0. [Install uv on your computer](https://docs.astral.sh/uv/getting-started/installation/#installation-methods).  
1. Click on "Use this template" on the top right of the page, and create your own repository. 
2. Clone your new repository to your computer. 
3. Run the command `uv sync` in a terminal. This will install all the python packages required by Zensical. 
4. To run the development server, use `uv run -- zensical serve`. The website will be viewable at [http://localhost:8000](http://localhost:8000). 
5. Once you are happy, push the changes to your own repository. 
6. Make sure GitHub Pages is enabled and configured to your liking for the repository. Set GitHub Pages to deploy from the `main` branch. The website should automatically be deployed to `<username>.github.io/<repository_name>`.

Parts that need to be changed in this template are marked with "TODO: " followed by a short description of what needs to be changed.
